Обзор Системы

TRIZEL Atlas предоставляет структурное отображение эпистемических отношений внутри системы TRIZEL. Это отображение показывает, как репозитории, артефакты и доказательства связаны в рамках трехуровневой структуры управления.

Только для отображения: Этот интерфейс представляет статические, предварительно проверенные данные без выполнения анализа или интерпретации.

Репозитории

Система TRIZEL работает на трех уровнях, каждый с назначенными репозиториями:

trizel-lab

Внутренний исследовательский репозиторий документации

Уровень-2: Презентация

trizel-site

Публичный репозиторий презентации (этот сайт)

Полномочия: Нет — только презентация

Эпистемические Отношения

Следующие отношения описывают структурные связи между компонентами системы. Они являются только описательными и не подразумевают исполнения или причинности.

Поток Управления

  • trizel-core → управляет → trizel-lab
    Управление Уровня-0 устанавливает политики для исследований Уровня-1
  • trizel-core → управляет → trizel-site
    Управление Уровня-0 устанавливает политики для презентации Уровня-2

Информационный Поток

  • trizel-lab → информирует → trizel-site
    Исследования Уровня-1 информируют содержание презентации Уровня-2
  • trizel-site → ссылается → trizel-core
    Презентация Уровня-2 ссылается на управляющую власть Уровня-0
  • trizel-site → ссылается → trizel-lab
    Презентация Уровня-2 ссылается на исследовательскую документацию Уровня-1
