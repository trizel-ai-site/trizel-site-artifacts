System Overview

The TRIZEL Atlas provides a structural mapping of the epistemic relationships within the TRIZEL system. This mapping displays how repositories, artifacts, and evidence connect within the three-layer governance framework.

Display-only: This interface presents static, pre-validated data without performing analysis or interpretation.

Epistemic Relationships

The following relationships describe the structural connections between system components. These are descriptive only and do not imply execution or causation.

Governance Flow

  • trizel-core → governs → trizel-lab

  • trizel-core → governs → trizel-site
    Layer-0 governance establishes policies for Layer-2 presentation

Information Flow

  • trizel-site → references → trizel-core
