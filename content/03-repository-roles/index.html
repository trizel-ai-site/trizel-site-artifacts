Repository Roles & Governance Mapping

Repositories within the TRIZEL system are assigned specific roles within the epistemic layer structure. They are not presented as flat or unordered lists.

Governance / Constitutional Authority

Establishes binding rules and policy. Source of Layer-0 authority.

Operational Governance

Implements and enforces Layer-0 policies. Manages artifact approval flows.

Execution / Analysis

Performs scientific analysis under Layer-0 authorization. Operates at Layer-1; not public-facing.

Publication / Site Surface

Presents approved artifacts to the public. Read-only; no execution authority. (trizel-site-artifacts)

Definition / Reference

Canonical definitions, schemas, and reference materials. Governance-pinned.

Archive / Snapshot

Immutable records of prior system states and approved releases. Provenance-anchored.

Authority Flow

Layer-0 (trizel-core) → Operational Governance → Layer-1 Execution → Layer-2 Presentation
