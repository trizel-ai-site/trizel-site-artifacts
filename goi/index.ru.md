## Назначение

Глобальный индекс наблюдений и производства (GOI) — это научный реестр, предоставляющий статические, проверяемые метаданные для экосистемы TRIZEL.

Этот индекс каталогизирует:
- **Наблюдения:** Источники данных, архивы и выходные данные мониторинга
- **Производство:** Научные результаты и результаты анализа
- **Репозитории:** Структура экосистемы и инвентаризация компонентов
- **Снимки:** Замороженные справочные пакеты с проверкой целостности

## Как Проверить

Все данные GOI статичны и проверяемы:

1. **Статический Источник Данных:** `/goi/data/global-index.json`
2. **История Коммитов:** Все обновления отслеживаются через коммиты Git в этом репозитории
3. **Валидация Схемы:** Все данные соответствуют опубликованным JSON-схемам в `/goi/schemas/`
4. **Без Выполнения во Время Выполнения:** Без JavaScript, без API, без динамической загрузки данных

## Источник Истины Данных

GOI рассматривает `/goi/data/global-index.json` как единственный источник истины. Этот файл содержит:
- Версия схемы
- Временная метка генерации
- Многоязычные заголовки
- Реестр источников
- Журнал наблюдений
- Журнал производства
- Реестр снимков

**Текущее Состояние:** Статическая структура-заполнитель — заполнение данных только через зафиксированные JSON-обновления.

## Техническая Классификация

- **Слой:** Слой-2 (Публичная Презентация / Артефакты)
- **Тип:** Статический справочный индекс
- **Авторитет:** Нет — только отображение и справка
- **Выполнение:** Ноль — все содержимое предварительно зафиксировано в статических файлах
- **Интеграция:** Нет — нет внешних вызовов, нет данных времени выполнения
