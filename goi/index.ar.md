## الغرض

مؤشر المراقبة والإنتاج العالمي (GOI) هو سجل علمي يوفر بيانات وصفية ثابتة وقابلة للتحقق لنظام TRIZEL البيئي.

يفهرس هذا المؤشر:
- **الملاحظات:** مصادر البيانات والأرشيفات ومخرجات المراقبة
- **الإنتاج:** المخرجات العلمية ونتائج التحليل
- **المستودعات:** هيكل النظام البيئي وجرد المكونات
- **اللقطات:** حزم مرجعية مجمدة مع التحقق من السلامة

## كيفية التحقق

جميع بيانات GOI ثابتة وقابلة للتحقق:

1. **مصدر البيانات الثابت:** `/goi/data/global-index.json`
2. **سجل الالتزامات:** يتم تتبع جميع التحديثات عبر التزامات Git في هذا المستودع
3. **التحقق من المخطط:** جميع البيانات تتوافق مع مخططات JSON المنشورة في `/goi/schemas/`
4. **لا تنفيذ في وقت التشغيل:** لا JavaScript، لا واجهات برمجة تطبيقات، لا جلب بيانات ديناميكي

## مصدر حقيقة البيانات

يتعامل GOI مع `/goi/data/global-index.json` كمصدر وحيد للحقيقة. يحتوي هذا الملف على:
- إصدار المخطط
- طابع زمني للإنشاء
- عناوين متعددة اللغات
- سجل المصادر
- دفتر أستاذ الملاحظات
- دفتر أستاذ الإنتاج
- سجل اللقطات

**الحالة الحالية:** هيكل عنصر نائب ثابت — ملء البيانات فقط عبر تحديثات JSON الملتزم بها.

## التصنيف التقني

- **الطبقة:** الطبقة 2 (العرض العام / القطع الأثرية)
- **النوع:** فهرس مرجعي ثابت
- **السلطة:** لا شيء — العرض والمرجع فقط
- **التنفيذ:** صفر — جميع المحتويات هي ملفات ثابتة ملتزم بها مسبقًا
- **التكامل:** لا شيء — لا مكالمات خارجية، لا بيانات وقت التشغيل
