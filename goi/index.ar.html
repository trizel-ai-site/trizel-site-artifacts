الغرض

المؤشر العالمي للمراقبة والإنتاج (GOI) هو سجل علمي يوفر بيانات وصفية ثابتة وقابلة للتحقق لنظام TRIZEL البيئي.

يقوم هذا المؤشر بفهرسة:

  • المراقبات: مصادر البيانات والأرشيفات ومخرجات المراقبة
  • الإنتاج: المخرجات العلمية ونتائج التحليل
  • المستودعات: هيكل النظام البيئي وجرد المكونات
  • اللقطات: حزم مرجعية مجمدة مع التحقق من السلامة

أقسام المؤشر

  • المراقبات

    سجل مصادر بيانات المراقبة والبيانات الوصفية

  • الإنتاج

    دفتر الأستاذ لمخرجات الإنتاج العلمي

  • المستودعات

    جرد وهيكل النظام البيئي

  • اللقطات

    سجل اللقطات المنشورة

كيفية التحقق

جميع بيانات GOI ثابتة وقابلة للتحقق:

  1. مصدر البيانات الثابت: /goi/data/global-index.json
  2. سجل الالتزامات: يتم تتبع جميع التحديثات عبر التزامات Git في هذا المستودع
  3. التحقق من المخطط: تتوافق جميع البيانات مع مخططات JSON المنشورة في /goi/schemas/
  4. لا يوجد تنفيذ في وقت التشغيل: لا JavaScript، لا واجهات برمجة التطبيقات، لا جلب البيانات الديناميكي
