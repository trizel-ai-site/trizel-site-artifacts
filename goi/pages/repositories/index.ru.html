Репозитории — Инвентарь Экосистемы

Инвентарь репозиториев каталогизирует структурные компоненты экосистемы TRIZEL, включая репозитории кода, сайты документации и компоненты инфраструктуры. Этот раздел предоставляет метаданные об организации репозиториев, классификации слоев, сопровождающих и зависимостях между репозиториями.

Проверка

Все записи репозиториев проверяются через:

  1. История коммитов Git в этом репозитории
  2. Проверка схемы JSON (схема: /goi/schemas/repository.meta.schema.json)
  3. Перекрестные ссылки на фактические URL репозиториев GitHub
  4. Проверка топологии слоев относительно архитектуры экосистемы
