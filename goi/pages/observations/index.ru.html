Наблюдения — Реестр

Реестр наблюдений каталогизирует источники данных, архивы и результаты мониторинга из экосистемы TRIZEL. Этот раздел предоставляет статические метаданные о репозиториях наблюдений, включая их роль, области, временной охват и официальные источники.

Текущий Статус: Метаданные наблюдений еще не опубликованы. Этот реестр будет заполнен, когда репозитории Слоя-1 создадут и зафиксируют свои файлы метаданных в /goi/data/sources/.

Проверка

Все записи наблюдений проверяются через:

  1. История коммитов Git в этом репозитории
  2. Проверка схемы JSON (схема: /goi/schemas/observation.meta.schema.json)
  3. Хэши целостности (когда доступны в метаданных)
  4. Официальные ссылки на источники, включенные в каждую запись
