## 验证

所有观测条目可通过以下方式验证：
1. 此存储库中的 Git 提交历史
2. JSON 模式验证
3. 完整性哈希（当元数据中可用时）
4. 每个条目中包含的官方来源引用

## 存储库角色分类

- **raw** — 原始数据收集和摄取
- **archive** — 长期数据保存
- **analysis** — 科学分析和处理
- **monitor** — 持续监控和警报
- **lab** — 实验和研究工作区
- **governance** — 政策和标准定义
