Снимки — Опубликованный Реестр

Реестр снимков каталогизирует замороженные эталонные пакеты, временные выпуски и архивные пакеты из экосистемы TRIZEL. Этот раздел предоставляет метаданные о неизменяемых снимках экосистемы, включая их содержание, идентификаторы версий, хэши целостности и официальные записи публикации.

Проверка

Все записи снимков проверяются через:

  1. История коммитов Git в этом репозитории
  2. Проверка схемы JSON (схема: /goi/schemas/snapshot.meta.schema.json)
  3. Криптографические хэши целостности для содержимого пакетов
  4. Проверка временных меток и аудиторский след публикации
