Instantanés — Registre Publié

Le registre des instantanés catalogue les paquets de référence figés, versions horodatées et paquets d'archivage de l'écosystème TRIZEL. Cette section fournit des métadonnées sur les instantanés immuables de l'écosystème, y compris leur contenu, identifiants de version, hachages d'intégrité et enregistrements de publication officiels.

Vérification

Toutes les entrées d'instantanés sont vérifiables via :

  1. Historique des commits Git dans ce dépôt
  2. Validation de schéma JSON (schéma : /goi/schemas/snapshot.meta.schema.json)
  3. Hachages d'intégrité cryptographiques pour le contenu des paquets
  4. Vérification de l'horodatage et piste d'audit de publication
