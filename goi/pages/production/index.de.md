## Zweck

Das Produktionsverzeichnis verfolgt wissenschaftliche Ausgaben und Analyseergebnisse, die vom TRIZEL-Ökosystem produziert werden. Dieser Abschnitt bietet eine statische Aufzeichnung von Produktionsausgaben, einschließlich ihres Typs, ihrer Quelle, zugehöriger Schnappschüsse und ihres Status.

## Datenquelle

Diese Seite zeigt Daten aus: `/goi/data/global-index.json`

**Aktueller Status:** Es wurden noch keine Produktionsausgaben veröffentlicht. Dieses Verzeichnis wird gefüllt, wenn wissenschaftliche Ausgaben produziert werden und ihre Metadaten in diesem Repository committet werden.

## Produktionstypen

Produktionsausgaben können umfassen:
- **Analyseergebnisse** — Wissenschaftliche Analyse und abgeleitete Datenprodukte
- **Verarbeitete Datensätze** — Bereinigte, transformierte oder aggregierte Daten
- **Berichte** — Zusammenfassungsberichte und Dokumentation
- **Visualisierungen** — Diagramme, Plots und grafische Darstellungen
- **Abgeleitete Produkte** — Sekundäre Datenprodukte, die aus primären Beobachtungen erstellt wurden

## Überprüfung

Alle Produktionseinträge sind überprüfbar durch:
1. Git-Commit-Historie in diesem Repository
2. Schnappschuss-Referenzen für Provenienz-Tracking
3. Quell-Repository-Attribution
4. Zeitstempel-Aufzeichnungen
