Global Observation Index (GOI)

The GOI is a multilingual catalog of verified scientific artifacts with full provenance tracking. All entries are static and governance-controlled.

Layer-2 Read-Only Presentation

This GOI index is a presentation-only view of artifacts committed to the repository. No runtime parsing, no dynamic content, no client-side JSON processing.

All artifact entries are populated from the committed global-index.json and rendered as static pages.
