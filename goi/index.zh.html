目的

全球观测与生产指数 (GOI) 是一个科学注册表，为 TRIZEL 生态系统提供静态、可验证的元数据。

该指数编目：

  • 观测：数据源、档案和监测输出
  • 生产：科学产出和分析结果
  • 存储库：生态系统结构和组件清单
  • 快照：具有完整性验证的冻结参考包

指数部分

  • 观测

    观测数据源和元数据注册表

  • 生产

    科学生产产出的分类账

  • 存储库

    生态系统清单和结构

  • 快照

    已发布快照注册表

如何验证

所有 GOI 数据都是静态且可验证的：

  1. 静态数据源：/goi/data/global-index.json
  2. 提交历史记录：所有更新都通过此存储库中的 Git 提交进行跟踪
  3. 架构验证：所有数据都符合 /goi/schemas/ 中发布的 JSON 架构
  4. 无运行时执行：无 JavaScript、无 API、无动态数据获取
