Уведомление Уровня-2

Этот репозиторий является Уровнем-2 (Только презентация/отображение). Он не содержит научного контента, не выполняет анализ и не имеет исполнительных возможностей. Вся власть исходит из trizel-core (Уровень-0), канонического репозитория управления.

Институциональный Обзор

TRIZEL управляется в соответствии с описанием в артефактах управления репозитория. Эта страница представляет собой статическую сводку, предназначенную только для презентации, для человеческих читателей и институциональных заинтересованных сторон.

Статус Управления

Уровень-2 (Управление)
ЗАВЕРШЕНО — заморожено. Реестр и артефакты управления поддерживаются в docs/ и защищены CODEOWNERS и защитой ветки.
Уровень-3 (Презентация)
Только статический контент. Здесь не выполняется автоматизация, скрипты, конфигурация DNS или домена.

Область и Ограничения

  • Идентификация публикации: trizel-admin (учетная запись публикации Уровня-2).
  • Разрешенные изменения для этого PR: только HTML и CSS, контент презентации.
  • Запрещенные изменения: docs/, .github/, защита ветки, рабочие процессы, DNS, пользовательские домены, файлы реестра.
  • В этом PR не включен JavaScript или динамический код.
