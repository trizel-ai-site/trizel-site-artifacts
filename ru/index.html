Уведомление об уровне-2

Этот репозиторий находится на уровне-2 (только представление/отображение). Он не содержит научного контента, не выполняет анализа и не имеет возможностей выполнения. Вся власть исходит от trizel-core (уровень-0), канонического репозитория управления.

Институциональный обзор

TRIZEL управляется, как описано в артефактах управления репозитория. Эта страница представляет собой статическую сводку только для представления, предназначенную для читателей и институциональных заинтересованных сторон.

Статус управления
