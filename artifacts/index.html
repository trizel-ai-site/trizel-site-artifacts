Verification Process

All published artifacts follow this mandatory verification process:

  1. Read files from Layer-1 authoritative source (READ-ONLY)
  2. Compute SHA-256 hash on Layer-1 (source)
  3. Copy files byte-for-byte to Layer-2 (target)
  4. Compute SHA-256 hash on Layer-2 (target)
  5. Verify 1:1 hash equality
  6. Publish with cryptographic proof

⚠️ Governance Requirement: Any hash mismatch constitutes a violation. Publishing is halted and the issue is reported immediately.
